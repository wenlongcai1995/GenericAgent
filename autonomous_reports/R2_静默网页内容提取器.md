# R2 静默网页内容提取器 — 能力树扩展报告

**报告时间**: 2026-04-24
**任务类别**: 能力树扩展
**前置依赖**: silent_search.py (刚创建)
**新增依赖**: trafilatura 2.0.0 (pip install)

---

## 一、背景

silent_search 可以在不打开浏览器的情况下搜索网页，但只能获取搜索摘要。为了完整获取信息，需要能静默打开链接并提取干净正文——R2 就是为了填补这个链路缺口。

## 二、产出物

| 文件 | 路径 | 说明 |
|------|------|------|
| `silent_fetch.py` | `../memory/silent_fetch.py` | 核心工具脚本 |
| `silent_fetch.py` | `./silent_fetch.py` | temp 副本 |
| `trafilatura` | pip 包 | 正文提取引擎 |

## 三、功能验证

| 测试项 | 结果 | 详情 |
|--------|------|------|
| JSON格式提取 | ✅ | 标题/日期/正文均正确，759字正文 |
| TXT格式提取 | ✅ | 1079字正文，比JSON模式含更多内容 |
| 中文页面 (BBC) | ⚠️ 超时 | 网络问题，非工具缺陷 |
| URL不可达 | ✅ 优雅降级 | 返回 success=False + 错误描述 |

## 四、API 设计

```python
from silent_fetch import silent_fetch

result = silent_fetch(url, output_format="json")
# 返回: {title, author, date, text, url, success, error}
```

支持三种输出格式：`text`（纯文本）、`markdown`（含链接）、`json`（含元数据）

## 六、链路价值

**silent_search → 搜索结果 → silent_fetch(提取正文) → 完整信息获取**

现在可以：搜索任何主题 → 选择感兴趣的结果 → 静默提取全文 → 分析和总结。完全不打开浏览器标签页，快速且无干扰。
