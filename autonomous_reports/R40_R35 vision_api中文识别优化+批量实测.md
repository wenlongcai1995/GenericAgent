## 核心发现

### 最佳Prompt (V2)
```
请仔细描述图片内容。注意：中文文字要逐字准确识别，
不要遗漏也不要写错别字；颜色要用标准名称描述
（红/橙/黄/绿/蓝/紫/黑/白/灰）。
```

## 待解决问题

1. **"vl→vi"字符混淆**: 行尾"vl"被误认为"vi"（R32同样问题，Ollama qwen3-vl的tokenizer对连续字母+行尾边界敏感）
2. **长文本截断**: num_predict=1024限制导致密集中文被截断
3. **V2不稳定**: test1(test基础中文)返回空的随机异常
4. **结构化prompt失败**: qwen3-vl对编号列表格式返回空
