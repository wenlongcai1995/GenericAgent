# inference.py - 统一推理接口（自动选择ollama/vision_api/llm_client）
# - 核心函数:
#   - inference.text(prompt, system, temperature, max_tokens) → str (文本推理，回退llm_client)
#   - inference.vision(prompt, image, ...) → str (视觉推理，自动回退OCR)

#   - inference.summarize(text, max_length) → str (文本摘要)
#   - inference.structured(prompt, output_schema, system) → dict (结构化输出)
#   - inference.status() → dict (ollama/vision可用性状态)
#   - inference.ollama_available → bool
#   - inference.vision_available → bool
# - 用法: from memory.inference import inference
# - 创建日期：R68-R73间迭代
# 注意: ollama自动检测；无ollama时回退llm_client/vision_api
